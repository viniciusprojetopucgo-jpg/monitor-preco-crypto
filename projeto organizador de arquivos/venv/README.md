# 📂 Organizador Inteligente de Arquivos

Script de automação desenvolvido em Python para organização sistemática de pastas (como Downloads ou Desktop). Ele categoriza arquivos automaticamente por extensão, movendo-os para subpastas específicas.

## 🎯 Objetivo do Projeto
Demonstrar habilidades em **automação de sistemas operacionais** e manipulação de arquivos/diretórios utilizando bibliotecas nativas do Python.

## 🚀 Tecnologias e Conceitos
* **Python 3.x**
* **Bibliotecas OS & Shutil:** Manipulação de caminhos e movimentação de dados.
* **Lógica de Mapeamento:** Uso de dicionários para gestão eficiente de extensões.

## 🛠️ Como Funciona
O script percorre a pasta configurada, identifica a extensão de cada arquivo e o move para uma pasta correspondente (Ex: `.pdf` -> Documentos, `.jpg` -> Imagens). Caso a pasta de destino não exista, o sistema a cria automaticamente.
