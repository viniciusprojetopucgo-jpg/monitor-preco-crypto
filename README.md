# 📈 Monitor de Preços Crypto em Tempo Real

Este projeto é um monitor de ativos digitais desenvolvido em Python que consome dados reais via API, realiza análise lógica de preços e dispara alertas automáticos no terminal. 

## 🚀 Tecnologias Utilizadas
* **Python 3.x**
* **Requests** (APIs REST)
* **Python-Dotenv** (Segurança de variáveis)
* **CoinGecko API**

## 🛠️ Arquitetura da Solução
A solução utiliza um arquivo `.env` para proteger dados sensíveis e um loop de monitoramento contínuo com tratamento de exceções.
